# 自动引入
>  这是一个测试功能，只会引入`phecda-server`暴露的`api`

类似`unplugin-auto-import`
## 快速开始

安装`unimport`后，就可以自动引入
```shell
npm i unimport -save-dev
```
启动后创建`p.d.ts`文件
在`tsconfig`中`include`中添加`p.d.ts`

```json
{
  "include": ["p.d.ts"]
}
```
会对所有`ts`文件引入
